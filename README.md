## Introduction

`go-stonfi` is a Go wrapper around the [Ston.fi API](https://api.ston.fi/swagger-ui/)

## Getting Started

### Installation

You can install the `go-stonfi` library using the following `go get` command:

```bash
go get github.com/itay747/go-stonfi
```
